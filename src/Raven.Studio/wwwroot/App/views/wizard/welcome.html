1. Welcome

The wizard will assist you with setting up your RavenDB server.
First, select whether to set up a new server/cluster or set the other nodes in an existing cluster configuration.

New Server/Cluster

When to use:
  • Setting up a new single Server
  • Setting up a new Cluster:
    • Wizard will configure the cluster and set up the first node.
    • After that, run this wizard again per node with 'Continue Setup'.
  • Optionally, only create a Setup Package for external setup

Continue Setup

When to use:
  • After completing a full wizard flow for 'New Cluster'
  • Select this option to set up the other nodes in the cluster
To disable this wizard and setup the server manually:
  • Open the settings.json file located in your RavenDB installation directory
  • Change the setup mode to None, e.g. "Setup.Mode: "None"
